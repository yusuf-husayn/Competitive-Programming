## 1) String Length (String طول الـ)

> [!NOTE] 
> **الفكرة التقنية (The Concept):**
>
> عشان نجيب عدد الحروف (طول) بتاع String.
> في لغة++ C، الـ `std::string` بيحتفظ جواه بمتغير بيسجل حجمه الفعلي. علشان كدا لما بننادي على دالة `()length` أو `()size`، البرنامج مش بيلف على الحروف كلها عشان يعدها (ودا كان هياخد وقت **$O(N)$**)، لا دا بيرجع القيمة المتخزنة علطول وبشكل مباشر.
>

> [!TIP] 
> **استخدام `getline` مكان `cin`**
> 
> لو الـ Input فيه مسافات (Spaces)، استخدام `cin >> S` هيقرا أول كلمة بس ويقف عند أول مسافة. علشان نضمن إننا نقرا السطر كله صح، لازم نستخدم `getline(cin, S)`.
>

> [!CAUTION] 
> **C++ Template:**
> ```cpp
> void solve() {
>     string S;
>     // قراءة السطر كله عشان منضيعش الحروف اللي بعد المسافات
>     getline(cin, S);
>     
>     // طباعة طول الـ String مباشرة
>     cout << S.length() << endl;
> }
> ```

## 2) Linear Search in 2D Array (البحث الخطي في 2D Array)

> [!NOTE] 
> **الفكرة التقنية (The Concept):**
>
> عشان ندور على عنصر X جوا مصفوفة ثنائية الأبعاد (2D Array) بالمرور التسلسلي. الخوارزمية دي بتعتمد إننا نلف على كل عناصر الـ Array صف بصف باستخدام حلقتين متداخلين (Nested Loops). أول ما نلاقي العنصر اللي بيساوي X، بنطبع `YES` ونقفل الدالة فوراً عشان نوفر وقت. لو الـ Loops خلصت كلها ومفيش حاجة، دا إثبات إن الرقم مش موجود، فبنطبع `NO`.
> 
> **الـ Complexity:**
> 
> - **Time Complexity:**
>
> بتاخد **$O(N x M)$** في أسوأ الحالات (Worst-case) لو العنصر مش موجود أصلاً أو قاعد في آخر خلية.
> - **Space Complexity:**
>
> بتاخد **$O(N x M)$** عشان محتاجين نخزن الـ Grid كلها في الميموري.
>

> [!TIP] 
> **مبدأ الخروج المبكر (Early Exit)**
>
> استخدام `;return` أو `;break` أول ما نلاقي اللي بندوَّر عليه دي من أهم حركات الـ CP. بتوفر وقت المعالجة وتمنع الجهاز إنه يلف على الفاضي في دورات بحث ملهاش لازمة.
>

> [!CAUTION] 
> **C++ Template:**
> ```cpp
> void solve() {
>     int N, M; cin >> N >> M;
>     int arr[N][M];
> 
>     // قراءة الـ 2D Array
>     for(int i = 0; i < N; i++) {
>         for(int j = 0; j < M; j++) cin >> arr[i][j];
>     }
> 
>     int x; cin >> x;
> 
>     // الـ Linear Search
>     for(int i = 0; i < N; i++) {
>         for(int j = 0; j < M; j++) {
>             if(arr[i][j] == x) {
>                 cout << "YES" << endl;
>                 return; // خروج مبكر أول ما نلاقي الهدف
>             }
>         }
>     }
>     cout << "NO" << endl;
> }
> ```
